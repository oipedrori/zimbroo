POLITIQUE DE CONFIDENTIALITÉ ET DE PROTECTION DES DONNÉES - ZIMBROO APP

La confidentialité de l'utilisateur ("utilisateur", "vous", "votre") est la priorité numéro un du Zimbroo App ("Zimbroo", "nous", "notre"). Cette politique décrit comment les informations sont collectées, utilisées, partagées et protégées lors de l'utilisation de notre application de gestion financière assistée par IA.

1. COLLECTE D'INFORMATIONS
Zimbroo collecte uniquement les informations nécessaires pour faire fonctionner le service :
A) Authentification : En vous connectant avec Google, nous collectons votre nom, votre adresse e-mail et votre ID d'utilisateur Google.
B) Transactions Financières : Nous collectons des données saisies manuellement ou via audio, incluant la description, la valeur, la date et la catégorie.
C) Intégration avec Notion : Nous collectons des jetons d'intégration et des IDs de bases de données que l'utilisateur configure explicitement.

2. UTILISATION DES INFORMATIONS
Les informations sont utilisées exclusivement pour :
A) Traiter les transactions et générer des rapports financiers personnalisés.
B) Personnaliser les suggestions via l'Intelligence Artificielle (Google Gemini).
C) Synchroniser les données entre vos appareils via Google Cloud (Firebase).
D) Garantir un accès sécurisé à votre compte.

3. STOCKAGE ET SÉCURITÉ
Toutes les données sont stockées sur l'infrastructure de Google Cloud Platform (Firebase) aux États-Unis d'Amérique, conformément aux normes internationales de sécurité. L'accès à vos données est limité à vous-même via votre authentification Google. Zimbroo ne partage, ne vend ou ne transfère vos informations financières à des tiers à des fins publicitaires ou de marketing.

4. PARTAGE AVEC DES TIERS
Nous fonctionnons via des APIs tierces pour des fonctionnalités spécifiques :
A) Google Gemini API : Les données textuelles sont traitées par l'IA pour la catégorisation et l'analyse.
B) Notion API : Uniquement lorsqu'elle est configurée, nous lisons les données de votre base sur Notion.
Nous ne partageons pas de données avec les organismes gouvernementaux ou les institutions bancaires.

5. VOS DROITS (RGPD)
Vous possédez les droits suivants en ce qui concerne vos données :
A) Accès : Le droit de consulter toutes vos données à tout moment.
B) Correction : Le droit de modifier toute transaction ou donnée.
C) Suppression : Le droit de supprimer définitivement votre compte et toutes les données associées. Une fois supprimées, les données ne peuvent plus être récupérées.

6. COOKIES ET DONNÉES LOCALES
Nous utilisons le stockage local dans le navigateur pour enregistrer les préférences de thème (clair/sombre), de langue et de devise, afin d'assurer une expérience utilisateur fluide.

7. MODIFICATIONS DE LA POLITIQUE
Nous nous réservons le droit de mettre à jour cette Politique de Confidentialité si nécessaire. Nous publierons toute modification sur cette page et informerons les utilisateurs via l'interface de l'application.

8. CONTACT ET SUPPORT
Pour toute question ou exercice de vos droits de confidentialité, contactez-nous via les canaux de support officiels de l'application.
